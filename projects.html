Artificial Intelligence / Machine Learning

Heart Disease Prediction Using Clinical Features

A machine learning case study that uses a Multi-Layer Perceptron Artificial Neural Network model to classify heart disease using clinical patient features from a UCI dataset.

  • Used 270 patient records with 13 clinical variables such as age, sex, chest pain type, blood pressure, and cholesterol.
  • Applied an MLP Artificial Neural Network with backpropagation for classification.
  • Achieved approximately 88.9% overall accuracy with good generalization across training, validation, and testing.
